# LSU-Wise-Research-Porosity-Calculation-Script
I developed a Python script that evaluates tree canopy porosity using silhouette images (black = tree, white=space). The image can be generated by the script or uploaded manually, and the program analyzes the gaps within the canopy to estimate how much wind can pass through the leaves.

I used cv2, numpy, and matplotlib dependencies for image processing, numerical analysis, and visualization. 
Cv2 was used to load and process images, detect tree regions, visualize results, and save processed image.
Numpy was used to analysie pixel values.
Matplotlib was used for visual analysis output. 

In order to change the image go to line 601 in the code and put the path of the image. 

When the script is ran there will be a prompt to choose 3 modes in order to crete a boundary around the trees. Mode 1 is the automatic detection mode which automatically detects the boundary without user input. Mode 2 allows the user to draw a box to define the boundary manually. Mode 3 allows the user to draw four lines to define the boundary manually
